# 使用指南

## 界面概览

- **顶部导航**：进入主页、视频播放、媒体库、设置、新番更新等功能区域
- **主页**：展示新番推荐、最近观看记录和个性化推荐内容
- **媒体库**：管理本地媒体文件和远程媒体服务器连接

## 播放功能

### 播放控制

- **基础控制**：播放/暂停、进度条拖拽、音量调节
- **播放速度**：支持多档倍速播放（0.5x - 2.0x）
- **全屏模式**：支持全屏播放和窗口化切换

### 弹幕与字幕

- **弹幕显示**：支持开启/关闭弹幕显示
- **字幕管理**：多字幕轨道选择和样式自定义
- **显示设置**：弹幕透明度、字体大小、显示区域等个性化调节
- **手动匹配**：如果发现弹幕匹配有误，播放界面可以手动匹配弹幕
- **发送弹幕**：登录弹弹play账号后，可在播放时发送弹幕互动

## 账号功能

### 弹弹play账号

- **登录账号**：在设置→账号中登录弹弹play账号
- **观看同步**：自动同步观看进度到弹弹play
- **弹幕互动**：发送弹幕参与社区讨论

### Bangumi同步

- **进度同步**：将本地观看记录同步到Bangumi收藏
- **评分同步**：同步评分到Bangumi个人档案
- **收藏管理**：自动更新Bangumi收藏状态
- **短评功能**：在动画详情页面撰写和查看Bangumi短评
- **访问令牌**：需要在 [bgm.tv](https://next.bgm.tv/demo/access-token) 创建访问令牌

### 播放历史

- **观看记录**：自动记录观看进度和历史
- **续播功能**：支持一键续播上次观看位置
- **历史管理**：查看和管理完整的观看历史记录

## 媒体库管理

### 本地媒体

- **文件扫描**：自动扫描指定目录下的视频文件
- **媒体信息**：显示文件详细信息和元数据
- **分类浏览**：按类型、日期等维度组织媒体内容

### NipaPlay 局域网共享

- **设备互联**：连接同一局域网内其他PC端 NipaPlay 设备
- **直接播放**：无需拷贝，直接播放其他设备上的视频
- **零配置**：无需安装额外服务器软件

### 远程服务器

详细的服务器连接和使用方法请参考 [媒体服务器整合](server-integration.md)。
